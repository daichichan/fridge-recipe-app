# 🧊 冷蔵庫ちゃん - AI レシピアシスタント

食材を入力するだけで、AI が完璧なレシピを提案するアプリケーション。可愛い冷蔵庫キャラがあなたの料理をサポート！

## ✨ 特徴

- 🤖 **AI レシピ生成** - Anthropic Claude API で最適なレシピを自動作成
- 📸 **自動画像表示** - 料理画像を自動検索して表示
- 💅 **可愛いデザイン** - パステルカラーの優しいUI
- 📱 **モバイル対応** - スマートフォンでも快適に使用可能
- 🌍 **世界中からアクセス** - Vercel で本番ホスト

## 🚀 クイックスタート

## 🛠️ 使い方

1. **食材を入力**
   - 「例：鶏肉、玉ねぎ、トマト...」の形式で入力

2. **Enterキーを押す**
   - 食材がタグとして追加されます

3. **「レシピを生成」をクリック**
   - AIが食材から最適なレシピを生成

4. **レシピ表示**
   - 料理画像、材料、作り方が表示されます
